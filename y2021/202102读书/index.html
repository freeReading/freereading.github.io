202102读书

2021-02-25

这个月五星的书有《苔丝》和罗素的《幸福之路》，欠一篇《幸福之路》的读后感，下个月完成。

《苔丝》

评分 ✨✨✨✨✨

本来是个俗套的韩剧故事会，但是因为这自相矛盾，互相对抗的时刻印上时代的刻痕就显得不一样，即便安琪已经厌恶了宗教的虚伪，也无法逃脱历史的束缚，即便他知道人生意义的大小，并不在于外界的变迁，而是在于主体的经验，他也还是一个习俗和成见的奴隶，更何况他也曾做了同样的事情，尤其他前期的哲学思考和他知道苔丝遭遇后的实际行动对比更让我觉得难受，男性凭什么就能如此理直气壮而女性为什么就能理应如此。

判断一个人的道德价值，应该根据他的本性，而不是根据他所做过的某一件事，一个人人格的美与丑，不在于达到的结果，而在于他的目的和冲动，对一个人的真实评价，不能只看他的历史，还要看他的将来。

最后写的真好：“那么就让她睡完觉吧！”

提前知道了故事梗概读起来就是一场煎熬，带着不可避免的宿命，苔丝一步步走向死亡。想起再往前三十年雨果笔下《悲惨世界》中的芳汀，第二次读的时候也是这般感觉，一个纯洁善良的女孩每一个新的段落都向毁灭更近一步。而差不多同时期美国人笔下《嘉莉妹妹》中的嘉莉，差不多的经历却最后功成名就，美国梦名不虚传。

书里苔丝的女性朋友们太善良了，要放到现在，可能苔丝还没碰到安琪就已经被人们吃的骨头都不剩了。

比如伊丝差点儿笑死我：今天他抱我的时候，我满以为他会吻我，于是我安安静静地趴在他的胸口，盼了一遍又一遍，身子动也没动。可是到头来他还是没有吻我。我再也不想在这儿待下去了。我要回老家去。

最喜欢苔丝挤牛奶的那一段，奶牛场还对每头牛命了名：矮胖子、幻想、高贵、薄雾、老来美、早美、整洁、洪亮，想起小时候去挤牛奶，手又小又没劲，捏了半天也挤不出来气的哭，好在那头牛老实，没有一脚把我踢飞。

《秋园》

评分 ✨✨✨

我能理解，但没有引起共鸣，读的时候想的很复杂，甚至有一些腻歪，那个年代占大多数的穷人不都过的这样的日子吗，姥姥比秋园难多了。

《哲学知识》

评分 ✨✨✨

相对简单一些，还是喜欢哲学史这样的入门书。

现象到实在 -> 物质的存在 -> 物质的本性 -> 观念论 -> 亲知和描述的知识 -> 归纳 -> 一般原则的知识 -> 先验知识 -> 共相的世界 -> 共相的知识 -> 直观知识 -> 真和假 -> 哲学的限度 -> 哲学的价值

《幸福之路》

评分 ✨✨✨✨✨

正好在断断续续看着耶鲁的公开课《心理学导论》，如何在自己脑袋中的混沌抽出几个线头是个不错的游戏。

对内多分析但也不要把一切心理失调归结为精神分析，对外多扩张领域培养兴趣，对不能理解之事多探究原因，适当建起和外界的屏障，对所谓的道德传统舆论不要那么在意，放轻松，爱他人。

PS：黄菡版本要比傅雷的版本翻译的好。

《把自己作为方法》

评分 ✨✨✨✨

自己很大程度上又是社会制度的产物，所以把自己当做方法其实还是把社会当做方法，所以不要被题目”诓骗“了。

项飙说”我们关心的是世界，不是自己“也是全书的支撑，但我关心的只有自己以及和自己发生联系的事物，不是世界，世界对于我来说太大太遥远了，慢慢不再喜欢熊培云就是因为他说的太空虚了，能解决我什么问题呢。这本书带给我些思考，但离自己还是有些远了。

《仿制药的真相》

评分 ✨✨✨✨

在看这本书之前，我以为药就是药，不区分仿制药还是品牌药，是药就应该发挥应该的疗效（除了假药），包括前几年《我不是药神》其实也没仔细理解电影的底层逻辑，光顾着emotion了。看完这本知道我浅薄了，仿制药并不比仿制电子产品更可靠。看来全球都运行着同样的宇宙法则，品牌药公司，美国药监局，仿制药公司，美国司法部门，还有我们的三氯牛奶。

《文学体验三十讲》

评分 ✨✨✨✨

如果现在，阅读的能力仍是你生命中的一部分，那你就像带着护身符的幸运儿。人类的阅读能力是进阶的，在儿童时代，大家一开始都为认识字母表而高兴，接着为可以读个句子而开心，接着可以读点儿童话故事了。那些后来没有得到阅读灵性召唤的人，他们的注意力转移到新闻报道或者报纸上的商业版块去了。最后还剩下一小部分人持续地着迷于字词。他们得到过一种狂喜而具有魔力的召唤，在获得阅读能力之后，他们并没有就把它当成一种工具，而是完全掉入书的王国，并一步一步去挖掘这个世界是多么博大精深。最开始，他们认为这是小孩儿的漂亮花园，有郁金香编织的床和金色的小池塘，而现在这个花园变成了一个公园，变成了一个景观，变成了地球的一部分。这个世界，有着天堂般的象牙海岸，它充满诱人的狂喜和常看常新的花瓣。并且，昨天看起来还像花园、公园或者热带丛林，今天明天，它看起来更像寺庙了，有成千的门厅和庭院，各个民族各个年代的精神财富，都在等待着被唤醒，组成一支复调的合唱。对于真正的读者来说，在书的无尽世界里，每个人都能在其中找到自己的一部分，穿越丛林有一千个可以抵达的地方，每一个都指向全新的开始，每一步都指向开放的道路。
